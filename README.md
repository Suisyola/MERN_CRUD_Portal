<H1>MERN-streams</H1>

There are 2 projects,

1. streams-backend
2. streams-frontend

<br/><br/>

<h2>How to start the project</h2>

<h3>streams-backend</h3>

1. modify the credentials for MongoDB in the file '.env'
   <br/>
2. issue 'node server | node my-transport-process.js'

<br/><br/>

<h3>streams-frontend</h3>

1. issue 'npm start'
